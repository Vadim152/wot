# Переключатель света прицела

Мод для «Мир танков» от Lesta, который позволяет переключать цвет прицела
правой кнопкой мыши и настраивать цвета через стандартный интерфейс настроек
модов.

## Установка

1. Скопируйте содержимое папки `mods` в одноимённую директорию клиента игры
   (`<путь_к_игре>/mods/<версия_клиента>/`).
2. Запустите игру. В ангаре откройте список модов и нажмите кнопку настроек
   мода «Переключатель света прицела».

## Использование

- В настройках мода укажите базовый и альтернативный цвет прицела.
- Во время боя нажмите правую кнопку мыши, чтобы переключить цвет.
- Повторное нажатие вернёт предыдущий цвет.

Настройки сохраняются между сессиями и применяются автоматически при входе в
бой.
